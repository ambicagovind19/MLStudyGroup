## 📄 How to Contribute 

Clone the repository to your local machine by running
```
git clone https://github.com/ambicagovind19/MLStudyGroup
cd MLStudyGroup
```
Commit on a separate branch as follows
```
git checkout -b weekX-update
```
Add your contributions, and commit changes:
```
git add WeekX/
git commit -m "I am the cookie monster"
git push origin weekX-update
```
